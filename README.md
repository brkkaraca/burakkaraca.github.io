# burak-karaca-site

Vite + React + Tailwind tabanlı kişisel web sitesi.

## Yerelde çalıştırma
```
npm install
npm run dev
```

## Build
```
npm run build
```

## GitHub Pages
- Repo adı: `burakkaraca.github.io`
- Settings → Pages → Source: **GitHub Actions**
- `main` branch'e push ettikten sonra otomatik deploy.
